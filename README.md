# Aldi-Store-Data-Anaysis (создание интерактивных дэшбордов для мониторинга с использованием MS Excel)
## Цель проекта
Магазин Aldi хочет составить  отчет о продажах за 2021, 2022, 2023 годы. Чтобы владелец
магазина Aldi мог понимать своих клиентов и увеличить продажи в 2024 и 2025 годах.
Магазин Aldi хочет подготовить отчёт о продажах за 2021–2023 годы с целью изучения клиентского поведения и разработки мер по увеличению продаж в 2024–2025 годах.
